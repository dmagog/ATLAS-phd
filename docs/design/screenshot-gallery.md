# Screenshot Gallery — для слайдов защиты

> **Версия:** 0.1 (2026-05-07)
> **Назначение:** аннотированная галерея скриншотов всех ключевых экранов с привязкой к шагам [`demo-script.md`](demo-script.md) и тезисам диссертации. Каждый скриншот сопровождается текстом «что показывает» и «какой тезис поддерживает» — для использования в слайдах.
> **Контекст:** [`docs/design-roadmap.md`](../design-roadmap.md) Фаза 6.

---

## Как пользоваться

1. **Запуск demo-стенда:** `./scripts/seed_demo.sh` (Phase 6.4 orchestrator).
2. **Логин под нужной ролью** (см. таблицу аккаунтов в README seed_demo.sh).
3. **Снять скриншот** в браузере (1440×900 рекомендуемое разрешение, dark mode по умолчанию). Сохранить в `docs/design/screenshots/after/<screen>.png`.
4. **Скопировать аннотацию** из соответствующего раздела ниже в подпись слайда.

Скриншоты `before/` сохранены не были (Phase 1 — блокер расширения Chrome). Для защиты используются только скриншоты «после» — сравнение «до/после» реализовано через текстовый аудит ([`ux-audit.md`](ux-audit.md)).

## 6. Шаг 5 — Supervisor heatmap (`/supervisor`)

**Файл:** `screenshots/after/06-supervisor-heatmap.png`
**Роль:** supervisor (`vasiliev@optics.demo`)
**Что показывает:**
- 4 hero-cards: студентов активных (12), попыток завершено (~64), топиков с fail-rate ≥ 50% (2), порог приватности (3.0).
- Per-topic таблица с реальными топиками (Дифракция Френеля и Фраунгофера: 67% fail; Поляризация: 50%; Интерференция: 43%; Полное отражение: 40%; Линзы: 33%; Принципы Ферма и Гюйгенса: 28.5%).
- Список студентов с privacy mask: 7 visible (Иванов, Петрова, Козлова, …) + 5 anonymized («Аспирант #3», «Аспирант #5», …).

**Тезис:** кафедральный режим работает. Научрук видит progress по топикам и общий список студентов, но текст ответов скрыт. Студенты, не давшие явного opt-in, видны только как «Аспирант #N».

**Демо-сценарий:** «Логинюсь как научрук. Вижу heatmap по топикам — самая проблемная тема — дифракция Френеля, 4 из 6 студентов ниже порога. Кликаю на ячейку — drill-down показывает только агрегатную статистику. Текст ответов студента я не вижу — это privacy by design, M5».
