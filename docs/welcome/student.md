# Welcome — аспирант

ATLAS — обучающая среда для подготовки к кандидатскому минимуму. Не аттестационный сервис, а помощник для самопроверки. Реальный экзамен сдаётся вне системы; здесь вы готовитесь к нему: задаёте вопросы по корпусу, проходите self-check, видите свой прогресс по программе.

## Регистрация

Научрук или методист отправит вам инвайт-ссылку вида:

```
https://atlas.ваша-кафедра/invites/<code>
```

Откройте её, заполните форму:
- email
- пароль
- галочка «согласие на обработку данных» — **обязательна** (без неё регистрация не пройдёт)

После регистрации вы автоматически залогинены.

## Что доступно

### 1. Q&A по программе
Задавайте вопросы по теме экзамена. Система ищет ответ в материалах вашей кафедры (Born&Wolf, Матвеев, Yariv и др.) и отвечает с цитатами на конкретные страницы.

Пример:
> «Что такое дифракция Френеля и чем отличается от Фраунгофера?»

Если вопрос off-topic (вне корпуса) — система **честно откажет** с пометкой «недостаточно данных», вместо того чтобы фантазировать.

### 2. Self-check по теме
Запустите тест по конкретной теме (билету). Получите 3-5 вопросов (multiple-choice + open-ended), оценку с разбором по 4 критериям:
- correctness (корректность фактов)
- completeness (полнота)
- logic (логика рассуждения)
- terminology (правильная терминология)

После сдачи — детальный feedback и общий балл от 0 до 5.

### 3. Дашборд прогресса
Видите по программе:
- какие темы сданы и с каким баллом
- какие темы не пройдены или требуют пересдачи
- ваши слабые места по error_tags (incomplete / logic_gap / terminology / factual_error)

## Privacy и видимость для научрука

По умолчанию ваши результаты **анонимизированы** в общегрупповой статистике. Научрук видит только агрегаты («N% группы провалили этот билет»), но **не кто именно**.

Если хотите, чтобы научрук мог видеть ваш индивидуальный профиль (для более точных консультаций) — переключите opt-in:

```
POST /me/visibility
{"visibility": "show-to-supervisor"}
```

Можете в любой момент снова закрыть видимость:
```
POST /me/visibility
{"visibility": "anonymous-aggregate-only"}
```

Каждое изменение видимости логируется (вы видите время в `/me`). Каждый просмотр вашего профиля научруком тоже логируется в audit-trail.

## Удаление аккаунта

Отправьте запрос tenant-admin'у — ваши данные будут удалены, попытки в self-check останутся в анонимизированной форме (для общегрупповой статистики), но без вашего email.
