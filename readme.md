# ATLAS

**ATLAS (Assistant for Technical Learning & Attestation Support)** — проект интеллектуального ассистента для подготовки к кандидатскому минимуму по технической специальности.

## Что делает система
- отвечает на учебные вопросы с опорой на базу материалов (RAG);
- показывает источники ответа (файл, раздел/страница, фрагмент);
- проводит самопроверку (тестовые и открытые вопросы);
- оценивает открытые ответы и дает рекомендации.

## Формат MVP
- каналы: Telegram-бот + web-чат;
- языки: русский и английский;
- роли: администратор и пользователь;
- запуск: локальный.
